Static Typing

En af de primære funktioner i TypeScript er Static Typing. Herunder vil jeg forklare hvordan man anvender Static Typing, samt forskellen på Static og Dynamic typing.

Static vs. Dynamic Typing

JS: Dynamic
I JavaScript anvender vi Dynamic Typing. Det betyder, at variabler kan holde værdier af enhver type, og at disse typer kan ændrer sig når programmet kører. Selvom dette er fleksibelt, kan det føre til runtime-errors!
Eksempel på Dynamic Typing:
                        
                            let message = 'Hello World!';
                            message = 12;
                            console.log(message);  //Output: 12
TS: Static
TypeScript håndhæver type constraints, når programmet kører, som kan reducerer mængden af runtime-errors. Man kan angive typer for bl.a. variabler og funktionsparametre.
Eksempl på Static Typing:
                        
                            let message: string = 'Hello World!';
                            message = 12; //Fejl: Typen 'number' kan ikke tildeles til typen 'string'
                            console.log(message); //Output: 'Hello World!'

Type Annotationer

Type Annotationer bruges til at specificere typen af en variabel, et funktionsparameter eller en returværdi. De annoteres på følgende måde:
Variabler:
                    
                        let count: number = 7;
                        let username: string = 'Mathias';
                        let isAdmin: boolean = true;
Funktionsparametre og returværdi:
                    
                        function greetUser (name: string): string { //Første annotering definerer parameter typen, anden annotering returværdiens type
                            return `Hello, ${name}!`;
                        }

                        console.log(greetUser('Mathias')); //Output: 'Hello, Mathias!
                        console.log(greetUser(123)); //Fejl: Argument af type 'number' kan ikke tildeles til parameter af type 'string'

Type Inference

TypeScript kan godt automatisk udlede typer baseret på de tildelte værdier, som kan være med til at reducere behovet for eksplicitte type annotationer
Eksempel:
                    
                        let count = 5; //Udledes som typen 'number'
                        count = 'fem'; //Fejl: Typen 'string' kan ikke tildeles til typen 'number'
