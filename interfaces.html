Interfaces

Man anvender interfaces til at definere strukturen af et objekt

Eksempel på interface:

                
                    interface Cat {
                        name: string;
                        age: number;
                        isCute: boolean;
                    }

                    let cat: Cat = {
                        name: 'kiwi',
                        age: 2,
                        isCute: true
                    }
I dette eksempel definerer 'Cat' interface at et hver objekt af typen 'Cat' skal have:
Et 'name' som er af typen string
En 'age' som er af typen number
En 'isCute' af typen boolean

Extending

Man kan extende andre interfaces med interfaces. Dette giver muligheden for at bygge mere komplekse interfaces udfra en simpel en. Man kan sammenligne det med at lave en udbygning på et hus.
Eksempel på extending:
                
                    interface Cat {
                        name: string;
                        age: number;
                        isCute: boolean;
                    }

                    interface catRace extends Cat{
                        race: string;
                    }

                    let cat: catRace ={
                        name: 'pandekage',
                        age: 3,
                        isCute: true,
                        race: 'munchkin'
                    }

Declaration Merging

Man kan merge interfaces. Hvis man definerer to interfaces med samme navn, vil Typescript kombinerer dem til et enkelt interface
Eksempel på merging:
                
                    Interface Cat {
                        name: string;
                        age: number;
                        isCute: boolean;
                    }

                    interface Cat {
                        race: string;
                    }

                    let myCat: Cat = {
                        name: 'kakao',
                        age: 4,
                        isCute: true,
                        race: 'British Shorthair'
                    }
