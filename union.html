Union Types

Union Types i TypeScript giver mulighed for at definere en variabel som indeholder mere end én type værdi. Det er nyttigt da det giver mere fleksibilitet samtidigt med at type-sikkerheden bevares.

Definering med Union Types

Eksempel på hvordan man definerer en variabel med mere end én type:
                
                    let værdi: string | number;
Her kan 'værdi', altså enten være en 'string' eller et 'number'.

Brug af union types

Man kan anvende union types til at tillade en variabel at acceptere forskellige værdier.
Eksempel:
                
                    let værdi: string | number;

                    værdi = 'hej'; //Dette er tilladt
                    værdi = 12; //Dette er også tilladt
                    værdi = true; //Dette vil forårsage en fejl, da boolean ikke er en tilladt type i vores definition

Union types i funktioner og arrays

Union type i en funktion:
                
                    function printId(id: number | string) {
                        console.log('Dit ID er:' + id);
                    }

                    printId(101); //Dette er tilladt
                    printId('202'); //Dette er også tilladt
                    printId(true); //Dette vil forårsage en fejl, da boolean ikke er en tilladt type i vores definition af id
Union type i et array:
                
                    let array: (number | string)[] = [1, 'to', 3, 'fire'];
